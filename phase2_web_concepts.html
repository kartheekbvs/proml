How the Web Works

HTTP Protocol: Think of it as a set of rules for ordering pizza. You (the browser) call and say "Give me a Margherita" (GET request). The server checks if it has one and gives it to you (200 OK) or says it's out (404 Not Found).

MODULE 2.2

REST APIs & CRUD

Nouns for resources, Verbs for actions.

  • POST /users: Create
  • GET /users/1: Read
  • PUT /users/1: Update
  • DELETE /users/1: Delete
MODULE 2.3

WSGI vs ASGI

WSGI: Synchronous (One at a time). Like a shop with one cashier.

ASGI: Asynchronous (Concurrent). Like a restaurant where one waiter takes many orders while food is cooking.

FastAPI uses Uvicorn (ASGI) to handle thousands of requests at once.
