MODULE 3.1

Flask Basics

Flask: A micro-framework. It's like a set of Legos; you start with a base and add only the pieces you need. It's the most PHP-like framework in Python.

python
from flask import Flask, jsonify

app = Flask(__name__)

@app.route("/")
def home():
    return jsonify({"message": "Hello from Flask!"})

if __name__ == "__main__":
    app.run(debug=True)
MODULE 3.4

Serving Your First ML Model

We use joblib to load pre-trained models.

python
import joblib
from flask import request

model = joblib.load("model.pkl")

@app.route("/predict", methods=["POST"])
def predict():
    data = request.json["features"]
    prediction = model.predict([data])
    return jsonify({"prediction": prediction.tolist()})
